## Try these examples

### Parallel Operations

```typescript
await run(async (step) => {
  const user = await step(() => fetchUser());

  const [posts, comments, followers] = await step.parallel({
    posts: () => fetchPosts(user.id),
    comments: () => fetchComments(user.id),
    followers: () => fetchFollowers(user.id),
  });

  return { user, posts, comments, followers };
});
```

### Conditional Logic

```typescript
await run(async (step) => {
  const user = await step(() => getUser());

  if (user.isPremium) {
    await step(() => loadPremiumFeatures());
  } else {
    await step(() => loadBasicFeatures());
  }

  return user;
});
```

### Retry with Backoff

```typescript
await run(async (step) => {
  const result = await step.retry(
    () => unreliableApiCall(),
    { attempts: 3, backoff: 'exponential', baseDelay: 100 }
  );

  return result;
});
```

### Race Condition

```typescript
await run(async (step) => {
  const winner = await step.race({
    primary: () => fetchFromPrimary(),
    fallback: () => fetchFromFallback(),
  });

  return winner;
});
```
